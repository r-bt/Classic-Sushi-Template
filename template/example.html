Rock! Paper! Scissors!

Using Python and Tkinter

Getting Started

  1. You'll need somewhere to write your code. We'll use the website https://repl.it
  2. There are three windows: The first is for files which are Python's Sprites
  3. Let's think about what we're going to be building.

    Tkinter is a library for Python which allows us to do new interesting things.
    Specifically we can now create GUIs or Graphical User Interfaces. 

    We're going to build a Rock, Paper Scissors game. We'll let the user pick their move and compare this to a randomly chosen computer move. Let's get going

Setting up the GUI

  1. First we need to tell Python to use the Tkinter library
    import tkinter
  2. Next we need the window. Go ahead and give it a title also
    window = tkinter.Tk()
    window.title("Rock! Paper! Scissors!")
  3. Last step before running
    window.mainloop() This is the last line of your script
